                        TUI QUICK START GUIDE
═══════════════════════════════════════════════════════════════════════════════

┌─ RUN THE DEMO ──────────────────────────────────────────────────────────────┐
│                                                                              │
│   $ python utils/tui_live.py                                                │
│                                                                              │
│   Shows simulated attention/cache visualization (no model needed)           │
│   Press Ctrl+C to exit                                                      │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

┌─ RUN WITH REAL MODEL ───────────────────────────────────────────────────────┐
│                                                                              │
│   $ uv run python -m core.rollout \                                         │
│       --model_dir /path/to/model \                                          │
│       --env_name vision \                                                   │
│       --episodes 10 \                                                       │
│       --tui                          ← ADD THIS FLAG!                       │
│                                                                              │
│   Visualizes REAL attention, KV cache, and sampling during rollout          │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

┌─ WHAT YOU'LL SEE ───────────────────────────────────────────────────────────┐
│                                                                              │
│   ╔════════════════════════════════════════════════════════════╗            │
│   ║ STEP 5 │ R: 0.85 │ KL: 0.02 │ ⚡ 142 tok/s               ║            │
│   ╚════════════════════════════════════════════════════════════╝            │
│                                                                              │
│   DECODE LIVE                              Cache: 45/2048                   │
│   Tok[12]: "Paris" p=0.89 ←━━━ SAMPLING NOW ⚡                              │
│                                                                              │
│   ATTENTION FLOW (Last 6 Layers × 16 Heads)                                 │
│   L24  ░░▒█▓░░▒▓██░░▒░  │ ← Text self-attn                                 │
│   L25  ▒▓██░░▒░░▓█▒░░▒  │                                                  │
│   L26  ░▒▓█████▓▒░░░░▒  │ ← Vision→Text ⚡ ACTIVE                          │
│   ...                                                                        │
│                                                                              │
│   KV CACHE MEMORY                                                            │
│   L24: [████░░░░░░░] 45/2048  180KB  ⚡ WRITE                               │
│   ...                                                                        │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

┌─ VISUAL LEGEND ─────────────────────────────────────────────────────────────┐
│                                                                              │
│   Intensity:                                                                 │
│     ░ = 0.00-0.25  (low attention)                                          │
│     ▒ = 0.25-0.50  (medium)                                                 │
│     ▓ = 0.50-0.75  (high)                                                   │
│     █ = 0.75-1.00  (very high)                                              │
│                                                                              │
│   Flashing (5Hz):                                                            │
│     ⚡ ACTIVE    - Yellow: Computation happening                            │
│     ⚡ WRITE     - Green: Cache being updated                               │
│     ⚡ SAMPLING  - Cyan: Token being sampled                                │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘
